Когда учишься тому, как запрогать блог, но в туториалах видишь вот такое:

```jsx
{previous === false ? null : (
  <>
    {previous && (
      <Link to={previous.fields.slug}>
        <button className="previous-next-button">
          {previous.frontmatter.title}
        </button>
      </Link>
    )}
  </>
)}
{next === false ? null : (
  <>
    {next && (
      <Link to={next.fields.slug}>
        <button className="previous-next-button">
          {next.frontmatter.title}
        </button>
      </Link>
    )}
  </>
)}
```
